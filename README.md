# customer-product-analysis-dashboard
A Power BI project that includes customer and product analysis, including sales data.

* Overview
A dashboard application displaying total product sales quantities, return amounts, and revenue information by year, based on customer names

* Dashboard Features
Customer-level sales analysis
Yearly revenue and return tracking
Interactive filters and dynamic visuals

*Tools Used
Microsoft Power BI Desktop
Data modeling
DAX calculations

*Purpose
The goal of this project is to support data-driven decision-making by providing a clear and interactive view of customer purchasing behavior and product performance.
